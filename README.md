# Task Manager Fullstack 🚀

Fullstack task management app built with:

## 🛠️ Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL + TypeORM

## ✨ Features
- Add / Edit / Delete tasks
- Mark tasks as completed
- Search tasks
- Full CRUD API

## ▶️ Run locally

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
